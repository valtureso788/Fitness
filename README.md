# 🏋️ FitClub — Сервис фитнес-клуба

> Автоматизированная система управления абонементами, тренировками и посещаемостью фитнес-клуба.

## 📋 О проекте

Веб-приложение для автоматизации ключевых процессов фитнес-клуба:
- Онлайн-запись на тренировки
- Управление абонементами
- Учёт посещаемости
- Формирование отчётов

## 👥 Роли пользователей

| Роль | Возможности |
|:---|:---|
| **Клиент** | Просмотр расписания, запись на тренировки, покупка абонементов, история посещений |
| **Тренер** | Просмотр расписания, отметка посещаемости |
| **Администратор** | Управление расписанием, абонементами, статистика |

## 🚀 Запуск

1. Откройте `app/index.html` в браузере
2. Войдите с демо-данными:
   - Клиент: `anna@mail.ru` / `123456`
   - Тренер: `maria@mail.ru` / `123456`
   - Администратор: `admin@fitness.ru` / `admin`
